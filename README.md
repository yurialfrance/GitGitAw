# GitGit Aw

A free, open-source Git learning platform built for Filipino developers. Step-by-step lessons explained in Taglish — walang judgement, puro aral lang.

## What is this?

GitGit Aw is an interactive web app that teaches Git and GitHub from the ground up. It's designed specifically for Pinoy developers who want to learn version control in a way that feels familiar and approachable — no intimidating jargon, no assumptions about prior knowledge.

## Features

- **Taglish lessons** — concepts explained in a mix of Filipino and English for easier understanding
- **Interactive quizzes** — test your knowledge at the end of each lesson
- **Hands-on code examples** — real terminal commands with copy-to-clipboard support
- **Dark and light mode** — with accent color customization and reduce-motion support
- **Fully free** — no accounts, no paywalls, open source under Apache-2.0

## Getting Started (Local)

```bash
git clone https://github.com/yurialfrance/GitGitAw.git
cd GitGitAw/gitgitaw
npm install
npm run dev
```
